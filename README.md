# securityLibrary

[![Unit tests](https://github.com/UNICORE-EU/securityLibrary/actions/workflows/maven.yml/badge.svg)](https://github.com/UNICORE-EU/securityLibrary/actions/workflows/maven.yml)

This library provides low level utility classes for UNICORE: 
 - security helpers based on [CaNL](https://github.com/eu-emi/canl-java/)
 - HTTP server using [Eclipse Jetty 12](https://jetty.org/)
 - HTTP client using [Apache HTTP Client](https://hc.apache.org/httpcomponents-client-5.6.x/index.html)

as well as various utilities and data types.

It is used in all of the UNICORE (Java-based) servers and the commandline client.
